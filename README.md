# Lunite Docs
Website for the docs, lessons, exercises of The Lunite Programming Language.

### File Structure
- `index.html`: Homepage
- `registry.json`: All in one registry for docs/lessons/questions
- `docs/...`: Documentation
- `lessons/...`: Lesson .md files
- `exercises/...`: Exercises .html files
- `diy/...`: Do It Yourself questions .md files
