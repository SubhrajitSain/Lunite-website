## How to Build Lunite Executable

To build Lunite, you may run either `build.sh` (for Linux or MacOS) or `build.bat` (for Windows). Make sure you have Python installed, because the script requires `pip` to install `pyinstaller` via `pip install pyinstaller` (this is done automatically in either script). These commands come pre-packaged into the Lunite source code in the GitHub repository (but you can make your own too!).  

After you run either script, `pyinstaller` will be installed on your system, after which it will run the necessary commands to make an executable for you.
