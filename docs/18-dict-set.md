
## Dict Or Set?

Always remember this:

```lunite
let a = {}                   ~~ Defaults to dict if declared empty
let b = { "name": "Lunite" } ~~ Dict has `:` in it
let c = {1, 2, 3, 4, 5}      ~~ Set has comma separated values, no `:`
```
